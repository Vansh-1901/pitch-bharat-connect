# Karo Pitch – Demo Landing Page

This project is a demo landing page designed for the Karo Pitch Internship Assignment by KaroStartup.

## Overview
Karo Pitch is a platform where founders from across India, especially Tier-2 and Tier-3 cities, can pitch their startups directly to investors and raise funding.

## Features
- Modern startup-style landing page
- Hero section with animated galaxy background
- How it works section explaining the pitch process
- Startup categories founders can apply from
- Investor showcase
- Featured startup cards
- Final call-to-action encouraging founders to apply

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite
